1. Elevation Architecture in Practice

How the L0 to L4 surfaces stack contextually.

L0: Atmosphere (--surface-l0 / --surface-0) The deepest recess of the app shell. Used for outer window backgrounds.
L1: Operating Surface (--surface-100) The primary canvas. Workspaces, main data grids, and shell navigation live here.
L2: Elevated Cards (--surface-l2 / --card) Standard content containers, metrics, and segmented data blocks. Creates logical grouping without visual noise.
L2: Interactive Card Click the button below to trigger the L4 Modal overlay.

L3: Floating Context (--popover) Used for dropdowns, tooltips, and command palettes. Casts a larger shadow.
